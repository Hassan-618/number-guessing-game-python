# Number Guessing Game - Python

A simple number guessing game implemented in Python with both console and GUI versions.

## Project Overview

This project was created as part of CSC-44102 Assessment 2, demonstrating Git workflow and Python programming skills.

## Game Description

The Number Guessing Game challenges players to guess a randomly generated number within a specified range and number of attempts.

## Features

- Console-based gameplay
- Multiple difficulty levels (Easy, Medium, Hard)
- GUI interface using tkinter
- Attempt tracking and scoring
- Input validation and error handling

## Installation

1. Clone this repository
2. Ensure Python 3.x is installed
3. Run the game using one of the methods below

## How to Play

### Console Version
```bash
python game.py
```

### GUI Version
```bash
python gui_game.py
```

## Game Rules

- **Easy**: Guess a number between 1-50 (10 attempts)
- **Medium**: Guess a number between 1-100 (7 attempts)  
- **Hard**: Guess a number between 1-200 (5 attempts)
